FAQ Administrador

¿Puedo modificar mis datos personales?

Si, para esto debe acceder a su perfil. El perfil es lo referente a los datos de cualquier usuario, del cual constan cuatro elementos identificables:

  • RUT
  • Tipo
  • Nombre
  • Teléfono
  • Correo Eléctronico
Para acceder a este debe ir al siguiente enlace o bien ingresar desde el menú que se muestra a continuación y seleccionar la opción Ver Perfil

¿Puedo modificar mi contraseña?

Si, para esto debe acceder a Cambiar contraseña. En este panel se le permite realizar el cambio de su contraseña actual por una nueva. Para acceder a este debe ir al siguiente enlace o bien ingresar desde el menú que se muestra a continuación.

Mantenedor de usuarios

El usuario administrador puede agregar, modificar y ver el detalle de los usuarios del sistema. También puede dar o quitar acceso a los usuarios ya registrados. Para realizar esto debe inregsar a la sección de Usuarios en el menú que se muestra a continuación.

Lista de las acciones mencionadas de mantenedor de usuarios:

a) Agregar nuevos usuarios al sistema

Para agregar un usuario debe seleccionar la opción Nuevo y llenar los datos en la ventana emergente y luego seleccionar la opción de Guardar

b) Modificar usuarios del sistema

Para modificar un usuario, debe seleccionar uno de la lista, a continuación debe elegir la opción Editar y modificar los datos que estime necesarios en la ventana emergente. Finalmente debe seleccionar la opción de Guardar

c) Ver el detalle de los usuarios del sistema

Para ver los detalles de un usuario, debe seleccionar uno de la lista, a continuación debe elegir la opción Ver Detalle donde se desplegará una ventana con los datos del usuario seleccionado. Una vez que termine, debe presionar la opción Cerrar para salir de la ventana emergente.

d) Dar/Quitar acceso a usuarios del sistema

Para dar o quitar el acceso de un usuario, debe seleccionar uno de la lista, a continuación debe elegir la opción Habilitar/Deshabilitar con lo que se le cambiará el acceso al usuario. Este será deshabilitado si se encontraba habilitado y habilitado en caso contrario.

Mantenedores

En la sección Mantenedores se le permite realizar el ingreso y modificación de distintos datos requeridos por el sistema. Estos datos pueden ser:

  • Enfermedades
  • Grupos Sanguíneos
  • Tipos de Parto
  • Previsiones
  • Géneros
  • Tipos de Acompañante
  • Tipos de Residencia
  • Países
  • Regiones
  • Comunas
Para acceder a este debe hacer click sobre cada uno de estos en el menú que se muestra a continuación.
